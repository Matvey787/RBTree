# Red-Black Tree

## Правила

1) Каждый узел либо красный, либо чёрный.
2) Корень всегда чёрный.
3) Все листья чёрные.
4) Оба потомка красного узла — чёрные.
5) Любой путь от узла к его потомкам содержит одинаковое число чёрных узлов (чёрная высота).
